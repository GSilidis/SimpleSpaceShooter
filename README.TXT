PROJECT TITLE: Simple Horizontal Shooter
PURPOSE OF PROJECT: To learn how to use greenfoot
VERSION or DATE: 1.5 (29.05.2016)
HOW TO START THIS PROJECT: Open Test.gfar with greenfoot and press start
AUTHORS: GSilidis
USER INSTRUCTIONS: Shoot enemies to get points.
Enemies will shoot at you. If you loose all your health it's game over.
At certain points you'll encounter bosses(100pt., 200pt., 300pt., 350pt.)
They have more health and shoot faster than regular enemies.
Game ends after defeating 4th boss. Hovewer you can continue plaing in endless mode by presing "Run".
